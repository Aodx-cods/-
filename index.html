사용 전 안내

ZIP 파일명은 종이 오염o.zip, 종이 오염x.zip, 종이 오염0.zip 같은 형식을 사용할 수 있습니다. 영어 o와 숫자 0 모두 오염됨으로 인식합니다.

STEP 01

학습 데이터 불러오기

저장소에 올린 ZIP 파일 또는 내 컴퓨터의 ZIP 파일을 이용해 브라우저에서 직접 모델을 학습합니다.

ZIP 데이터셋 학습

품목과 오염도를 자동으로 분류하여 학습합니다.

내 컴퓨터에서 ZIP 여러 개 선택 예: 종이 오염o.zip, 비닐 오염x.zip
